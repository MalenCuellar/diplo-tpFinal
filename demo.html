Reveal.js

Framework para Presentaciones en HTML

Created by Hakim El Hattab and contributors

Le permite crear hermosas diapositivas interactivas usando HTML. Esta presentación le mostrará ejemplos de lo que puede hacer.

Slides

No codifica? No hay problema. Existe un editor visual con todas las funciones para que pueda crearlos, pruébelo en https://slides.com.

Slide Backgrounds

Utilice data-background="#dddddd" en una diapositiva para cambiar el color de fondo. Todos los formatos de color CSS son compatibles.

Image Backgrounds

<section data-background="image.png">

Tiled Backgrounds

<section data-background="./imagen.png" data-background-repeat="repeat" data-background-size="100px">

Video Backgrounds

<section data-background-video="video.mp4,video.webm">

... and GIFs!

Background Transitions

Different background transitions are available via the backgroundTransition option. This one's called "zoom". 

Reveal.configure({ backgroundTransition: 'zoom' })

Background Transitions

You can override background transitions per-slide. 

<section data-background-transition="zoom">
